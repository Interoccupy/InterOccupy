== Description ==

The PageLines reader is a javascript powered section ideal for use as an FAQ or quick documentation area.  

It adds a tabbed interface at left that is used to flip through various 'reader posts' which could consist of a FAQ question/answer or a document to explain something. 

== Features ==
* Tabs at left, content on right
* No page reload needed for content navigation
* Use full-width or content width
* Easily customize with CSS
* Custom-post-type driven
* Supports WordPress shortcodes
* Support images and media
